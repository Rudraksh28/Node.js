REST API - JSON

GET /users -- List all users  html document

GET /api/users -- List all users JSON format

GET /api/users/l - the user with ID 1

GET /api/users/2 — Get the user with ID 2

POST /api/users — Create new user

PATCH /api/users/l - Edit the user with ID 1

DELETE /api/users/l - Delete the User with ID 1
